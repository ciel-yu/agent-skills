# AI Capability Catalog

Use this catalog when generating a Product Spec to map features to the appropriate AI capabilities.

---

## Image

| Capability | Description |
|---|---|
| Image Generation | Generate images from text descriptions |
| Image Editing | Modify, enhance, or transform existing images |
| Style Transfer | Apply artistic styles to images |
| Image Recognition | Identify objects, scenes, and people in images |
| OCR | Extract text from images |
| Image Captioning | Generate text descriptions for images |
| Image Classification | Categorize and label images |

---

## Video

| Capability | Description |
|---|---|
| Video Generation | Generate video from text or images |
| Video Editing | Cut, splice, and apply effects |
| Video Analysis | Analyze video content, scenes, and actions |
| Video Summarization | Extract key frames or generate summaries |
| Video Captioning | Auto-generate subtitles |

---

## Audio

| Capability | Description |
|---|---|
| Text-to-Speech (TTS) | Convert text to natural-sounding speech |
| Voice Cloning | Replicate a specific voice's characteristics |
| Speech-to-Text (STT) | Transcribe speech to text |
| Speaker Recognition | Identify speaker identity, emotion, or language |
| Voice Commands | Recognize and execute spoken instructions |

---

## Intelligence

| Capability | Description |
|---|---|
| Text Generation | Generate articles, summaries, translations |
| Code Generation | Generate code from a description |
| Creative Writing | Generate stories, poems, and creative content |
| Text Optimization | Improve readability, style, or tone |
| Content Rewriting | Rewrite text while preserving meaning |
| Grammar Checking | Detect and correct grammatical errors |
| Personalized Recommendations | Recommend content based on user behavior |
| Smart Suggestions | Provide contextually relevant suggestions |
| Data Extraction | Extract structured information from unstructured data |
| Data Analysis | Analyze data and generate insights |

---

## Data Connectors

| Capability | Description |
|---|---|
| API Integration | Connect to external APIs to fetch data |
| Database Connection | Query data from connected databases |
| File Processing | Handle PDF, Excel, Word, and other formats |
| Real-time Data Streams | Process live data streams |
| Data Synchronization | Sync across multiple data sources |

---

## Quick-Match Lookup

| Requirement Type | Consider First |
|---|---|
| Image processing | Image Generation / Image Editing / Image Recognition |
| Video content | Video Generation / Video Analysis / Video Summarization |
| Voice interaction | STT / TTS / Voice Commands |
| Content creation & optimization | Text Generation / Text Optimization / Creative Writing |
| External data | API Integration / Database Connection / File Processing |

## Capability Combinations

- **Image Generation + Image Editing**: Generate then refine
- **STT + Text Generation**: Voice input, auto-generate content
- **Data Extraction + Data Analysis**: Extract structured data, then generate insights
- **Text Generation + Text Optimization**: Generate draft, then auto-polish
